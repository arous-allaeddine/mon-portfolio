# Portfolio — Alla-eddine Arous

Portfolio personnel de **Alla-eddine Arous**, Développeur Full Stack Senior (Java / Spring / Angular / Cloud) basé à Montréal.

🔗 **En ligne :** https://arous-allaeddine.github.io/mon-portfolio/

## Caractéristiques

- Page unique, autonome (un seul `index.html`) — aucune dépendance, aucun build.
- Bilingue **FR / EN** (bascule instantanée, préférence sauvegardée).
- Thème **clair / sombre** (bascule, préférence sauvegardée).
- Design moderne responsive : héros animé, timeline d'expérience, cartes de projets, barres de langues.
- Animations au défilement (IntersectionObserver) et respect de `prefers-reduced-motion`.
- HTML/CSS/JS vanilla — fonctionne tel quel sur GitHub Pages.
